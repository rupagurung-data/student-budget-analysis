# Student Budget Analysis

## Project Overview

This project analyzes student income, expenses, and savings using Python, Pandas, and Matplotlib. The goal is to identify spending patterns, compare savings across programs and countries, and visualize key insights.

## Dataset Description

The dataset contains information about students, including:
- Student ID
- Name
- Country
- Program
- Monthly Income
- Rent
- Food Expenses
- Transport Expenses
- Entertainment Expenses
- Other Expenses
- Tuition Fee
- Scholarship

## Data Cleaning

- Removed duplicate records
- Filled missing scholarship values with 0
- Standardized country and program names

## Analysis Performed

- Top 5 savers
- Average savings by program
- Average savings by country
- Highest average income by program
- Highest average rent by country

## Visualizations

- Savings by program
- Savings by country
- Savings distribution

## Tools Used

- Python
- Pandas
- Matplotlib
